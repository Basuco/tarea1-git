Tarea 1

Esta primera tarea consiste en familiarizarse un poco con herramientas que deben ser comunes para el desarrollo de cualquier proyecto. Para lograr esto, tendrás que seguir los siguientes pasos:

  1. Tener una cuenta en github.
  2. Realizar un fork en github de este repo: http://github/devlabs-usb/tarea-1.
  3. Clonar tu propio fork usando git clone.
  4. Bajarte a tu repositorio local el branch test de nuestro repositorio http://github/devlabs-usb/tarea-1.
  5. Hacer merge de ese nuevo branch a tu master
  6. Responder las preguntas que se encuentran en este documento, usando tu propia copia.
  7. Hacer push de este documento a tu repositorio cuando termines de rellenar las respuestas.
